Tags and HTML element
For creating HTML
Unit 5 - Web Pages 101

Tags are the words or letters surrounded by angle brackets < and >. HTML elements have a start tag, content in-between, and an end tag. An example is <*p>I like pie<*/p> without the asterisks.

Hyperlink
The a tag
Unit 5 - Web Pages 101

The <*a><*/a> tag can be used to create clickable links on a web page. An example of a functioning one would be <*a href="https://developer.mozilla.org/en-US/" title="MDN">click me!<*/a>, but without the asterisks. By adding target="blank" in the tag you can make the link open a new browser window on click.

Class (CSS)
Adding specificity to an element
Unit 5 - Web Pages 101

A class is an attribute applied to one or multiple HTML elements to add styling in the stylesheets. An element can have more than one class. A CSS example is:
.paragraph {
font-weight: bold;
}
In which whatever element is given .class, it'll have bold text. To give an element a class, you add class="className" to the HTML tag, like <*div class="paragraph"><*/div> without the asterisks.
